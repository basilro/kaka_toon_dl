# kaka_toon_dl

카카오웹툰(webtoon.kakao.com) 무료/기다무 회차를 자동으로 내려받는 FlaskFarm 플러그인.

## 주요 기능

- **설정** — 쿠키(로그인) 주입, 다운로드 경로, 회차당 처리량, 기다무/소장권 사용 여부,
  출력 포맷(webp 등), 프록시, 압축, 알림 웹훅 등을 설정한다.
- **연재 목록** — 카카오웹툰 요일별 연재작 전체를 한 화면에서 불러와, 요일 필터·검색으로
  원하는 작품을 골라 **체크 한 번으로 다운로드 스케줄에 등록**한다. 작품 코드를 일일이
  찾아 입력할 필요가 없다. 이미 등록된 작품은 표시되며 중복 등록되지 않는다.
- **수동 다운로드** — URL 또는 작품 코드로 특정 작품을 즉시 내려받는다.
- **진행 상황** — 현재 다운로드 진행 상태를 실시간으로 확인한다.
- **다운로드 이력** — 처리된 작품/회차 기록을 조회한다.

## 동작 방식

- 등록된 작품의 무료·기다무·소장 회차를 스케줄에 따라 자동 수집한다.
- 카카오웹툰 BFF API를 사용하며, 로그인 쿠키로 사용자별 보유 티켓/대기무료 상태를 반영한다.
- 연재 목록은 공개 timetable API(`section/v2/timetables/days`)에서 가져오므로 로그인이 필요 없다.
