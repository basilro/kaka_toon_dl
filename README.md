# 카카오웹툰 기다무 자동 다운로드 (kaka_toon_dl)

SJVA / FlaskFarm 플러그인. 카카오웹툰 기다무(기다리면 무료) 회차를 스케줄에 따라 자동 다운로드한다.

---

## 설치

SJVA 플러그인 관리 → GitHub URL 입력:
```
https://github.com/basilro/kaka_toon_dl
```

---

## 설정 항목

| 항목 | 기본값 | 설명 |
|---|---|---|
| **작품 목록** | (없음) | 한 줄에 하나 또는 `\|` 구분. 제목 검색, URL, content_id 숫자 모두 가능 |
| **쿠키 (JSON)** | (없음) | Cookie-Editor 익스텐션으로 export한 JSON. 카카오 로그인 쿠키 |
| **저장 경로** | (없음) | 다운로드 루트 폴더 경로 |
| **스케줄** | `0 */3 * * *` | cron 형식(3시간마다) 또는 분 단위 숫자 |
| **시작 시 자동실행** | Off | 켜면 스케줄러 등록. 분 단위 interval이면 재시작 직후 1회 즉시 실행 |
| **회차당 최대 다운** | `1` | 한 작품에서 1회 실행당 최대 다운로드 회차 수 |
| **기다무 대여권 사용** | On | 기다무 충전 대여권 자동 차감 |
| **보유 이용권 사용** | Off | 보유 중인 일반 이용권 사용 여부 |
| **이미지 출력 형식** | `webp` | 다운로드 이미지 포맷 (webp / jpg 등) |
| **유료화 공지 자동 다운** | Off | 매월 유료화/종료 예정 작품을 공지 감지 시 자동 다운로드 |
| **유료화 저장 하위 폴더** | `완결` | 유료화 자동 다운 시 사용할 서브폴더명 |
| **ZIP 압축** | Off | 회차 다운 완료 후 이미지 폴더를 ZIP으로 묶고 원본 삭제 |
| **프록시 사용** | Off | warproxy 등 프록시 URL 설정 시 활성화 |
| **쿠키 만료 웹훅** | (없음) | 쿠키 만료 감지 시 Discord/Slack 알림 URL |
| **다운로드 웹훅** | (없음) | 다운로드 완료 요약 알림 URL |

## 동작 방식

### 자동 다운로드
1. 작품 목록에서 기다무/보유 회차 확인
2. DB에 `completed` 기록이 없는 회차를 회차당 최대 설정 수만큼 다운로드
3. `partial` / `failed` 상태 회차는 **다음 실행 시 자동 재시도** (이전 폴더 삭제 후 처음부터)

### 회차 상태
| 상태 | 의미 |
|---|---|
| `completed` | 전체 페이지 다운로드 완료 — 이후 스킵 |
| `partial` | 일부 페이지만 성공 — 다음 실행 시 재시도 |
| `failed` | 다운로드 실패 — 다음 실행 시 재시도 |

---

## 기술 사항

- **DB**: `kaka_toon_dl.db` (플러그인 전용 SQLite, WAL 모드 자동 적용)
- **잠금**: 다른 플러그인과 독립 실행 — kaka_pe_dl 등이 돌아도 블록 없음
- **UI**: 작업 중에도 작업내역 페이지 조회 가능 (WAL 모드)
- **메타**: 다운로드 폴더에 `info.xml`(ComicInfo) + `cover.jpg` 자동 생성 — Kavita/Komga 호환
